Terminology

  • A Course is a container for several Topics.
  • A Post has a title, a discussion prompt or question, and zero or more Comments as replies.
  • A Comment is a reply to a Post. It contains text.


Component Gallery

App Framework

Container holding App Header along the top, Post List along the left side. Center content area will be filled by a component, depending on what action the user is trying to perform.

App Header

Top-of-page header. Contains controls related to the user or the course

Post List

Left sidebar. Contains a sortable list of posts, and quick controls for viewing a post, or doing one-click toggle actions on a post.

Post Display

Center content if the user is reading a post. Shows the post, its comments, and other controls related to this post or its comments.

Comment Create

Editor for creating a comment

Comment Display

Area for displaying a comment
